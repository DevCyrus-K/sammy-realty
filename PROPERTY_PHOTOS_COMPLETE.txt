🚀 QUICK START (7 STEPS)
================================================================================

1. npm install @supabase/supabase-js formidable

2. Create Supabase project at https://supabase.com
   - Create bucket: "property-photos" (PUBLIC)

3. Get credentials from Supabase (Settings > API):
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - SUPABASE_SERVICE_ROLE_KEY

4. Add to .env file

5. npx prisma migrate dev --name add_property_photos

6. Update src/pages/api/properties/upload.ts with real Supabase code

7. Integrate PropertyPhotoGallery into admin & PropertyPhotoDisplay into frontend
